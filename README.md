# AWS S3 Tagging Enforcer

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Python CLI tool to enforce required tags on S3 buckets (mock). Logs before/after for compliance audits.

## Features
- Custom tags via CLI.
- Counts tagged items.
- Simulates 100% governance (e.g., mandatory tagging for risks).
